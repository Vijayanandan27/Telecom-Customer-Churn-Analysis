# Telecom-Customer-Churn-Analysis
Customer churn is a major challenge in the telecom industry. This project analyzes customer data to identify key drivers of churn and provide actionable business recommendations.
📁 Dataset
7,043 customer records
Includes:
Demographics
Service subscriptions
Account & billing details
⚙️ Tech Stack
Python
Pandas (Data Processing)
NumPy
Matplotlib & Seaborn (Visualization)
🧹 Data Preparation
Merged customer & population datasets
Handled missing values
Removed duplicates
Created new feature: Age Group segmentation

📊 Key Analysis
1. Target Variable Analysis
Churn Rate: 26.5%
2. Demographic Insights
Senior customers show highest churn (~36.5%)
3. Service Analysis
Fiber Optic users churn more (~40.7%)
Month-to-Month contracts → highest churn (~45.8%)
4. Financial Insights
Churned customers have higher monthly charges
5. Root Cause Analysis

Top churn reasons:

Better offers (16.6%)
Better devices (16.7%)
Poor customer support (11.8%)
💡 Business Recommendations
Focus retention strategies on senior customers
Encourage long-term contracts
Improve customer service experience
Reassess pricing strategy
